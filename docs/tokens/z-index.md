# Z-Index Tokens

Z-indexes are used to stack components in a logical manner.

| Token                       | Value |
| --------------------------- | ----- |
| `--sl-z-index-default`      | 1     |
| `--sl-z-index-panel`        | 80    |
| `--sl-z-index-notification` | 90    |
| `--sl-z-index-dialog`       | 100   |
